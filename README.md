# BoraLucrar

O projeto Bora Lucrar é um site voltado para ajudar vendedores ambulantes a organizar melhor suas finanças. 
A ideia é oferecer uma ferramenta simples e acessível para registrar vendas, controlar gastos e acompanhar o lucro no dia a dia. Com isso, 
o vendedor consegue ter mais clareza sobre quanto realmente está ganhando, tomar decisões melhores para o negócio e evitar prejuízos,
mesmo sem ter conhecimento avançado em finanças.

Neste projeto vamos usar HTML, CSS e JavaScript para estruturar, estilizar e dar interatividade às páginas.O Front-end será a parte visível para o usuário,
enquanto o Back-end cuidará da lógica e do processamento dos dados. Assim, teremos um sistema completo, simples e eficiente.
